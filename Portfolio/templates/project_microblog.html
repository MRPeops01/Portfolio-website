Microblog is a lightweight web application designed for personal notes and short text updates. It provides a clean, intuitive interface to jot down thoughts, ideas, or daily reflections quickly.

Users can create, edit, and delete posts, allowing for easy organization and management of their personal notes. The app focuses on simplicity, keeping the experience fast and distraction-free.

Key features include user authentication, secure storage of posts, and responsive design for both desktop and mobile devices. The MongoDB integration ensures that notes are stored efficiently and reliably.

Built with Flask and MongoDB, Microblog demonstrates full-stack development skills, including dynamic database interactions, front-end design, and a smooth, user-friendly experience.
